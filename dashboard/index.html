itch-lob-engine — bench dashboard

A C++20 engine that parses the real NASDAQ TotalView-ITCH 5.0 binary protocol and reconstructs the full limit order book per symbol — this page charts its performance benchmarks (OrderBook vs. LadderBook, single- vs. multi-threaded replay).

Renders bench/results.csv, bench/results_threaded.csv and bench/ci_history.csv directly, client-side, with no build step. Regenerate the source data with cmake --build build --target bench && ./build/bench (and ./build/bench_threaded for the threaded numbers), then reload this page.
